# 面向低质量医学图像的退化识别、自适应增强与分割效果评价模型

## 1. 项目概述

本项目面向低质量眼底医学图像，研究不同图像退化类型、增强方法选择与血管分割效果之间的关系。

项目主线如下：

```text
医学图像数据
-> 构造低质量退化图像
-> 提取退化特征
-> 训练退化识别模型
-> 根据退化类型自适应增强
-> 医学结构分割
-> 图像质量与分割效果评价
-> 撰写论文并提交代码和数据
```

核心研究问题：

1. 不同退化类型下，是否应该采用不同的增强方法？
2. 自适应增强是否比固定增强更能提高医学图像分割效果？
3. 图像视觉质量提升是否一定会带来医学结构分割性能提升？

项目最终需要证明：

```text
自适应增强能够在不同退化条件下稳定提高 Dice 和 IoU，
并降低 Hausdorff Distance。
```

## 4. 阶段三：构造低质量医学图像

真实低质量医学图像通常没有对应的清晰标准答案，因此从原始清晰图像出发，人工生成可控退化图像。

### 4.1 退化类别

```text
0：正常图像
1：噪声退化
2：模糊退化
3：低对比度退化
4：混合退化
```

### 4.2 退化参数

```text
高斯噪声：sigma = 5, 15, 30
高斯模糊：kernel = 3, 5, 7
低对比度：alpha = 0.4, 0.6, 0.8
```

混合退化包括：

- 噪声 + 模糊；
- 噪声 + 低对比度；
- 模糊 + 低对比度；
- 噪声 + 模糊 + 低对比度。

### 4.3 数学模型

退化过程表示为：

```text
I_d = D_k(I) + n
```

其中：

- `I` 为原始清晰图像；
- `I_d` 为退化图像；
- `D_k` 为第 `k` 类退化算子；
- `n` 为随机噪声。

### 4.4 阶段产出

- 正常、噪声、模糊、低对比度和混合退化图像；
- 每张退化图像对应的退化标签；
- 退化参数记录表 `degradation_manifest.csv`。

## 5. 阶段四：提取退化识别特征

对每张退化图像提取可解释的统计特征，不直接使用深度学习模型。

### 5.1 建议特征

```text
灰度均值
灰度标准差
图像熵
Laplacian 方差
边缘密度
局部对比度
高频残差标准差
暗像素比例
```

### 5.2 特征含义

```text
Laplacian 方差较低 -> 图像可能模糊
高频残差较大       -> 图像可能存在噪声
灰度标准差较低     -> 图像可能低对比度
边缘密度变化       -> 图像结构可能发生退化
```

### 5.3 输出文件

输出 `degradation_features.csv`，格式示例：

```text
image_id,split,mean,std,entropy,lap_var,edge_density,noise_score,label
```

## 6. 阶段五：训练退化识别模型

使用统计特征训练退化分类器。

### 6.1 对比模型

- Random Forest；
- SVM。

模型输入：

```text
x_i = 第 i 张图像的统计特征向量
```

模型输出：

```text
c_i = 第 i 张图像的退化类别
```

### 6.2 数据划分

- DRIVE training 生成的退化图像用于模型训练；
- DRIVE test 生成的退化图像用于最终测试；
- 禁止使用测试集调节模型参数。

### 6.3 评价指标

```text
Accuracy
Precision
Recall
Macro F1-score
混淆矩阵
```

由于不同退化类别样本数量可能不完全一致，应重点关注 Macro F1-score，而不是只看 Accuracy。

阶段目标：

```text
验证统计特征能否有效区分正常、噪声、模糊、低对比度和混合退化。
```

若 Random Forest 的测试集 Macro F1-score 更高且更稳定，则将其作为后续自适应增强的主模型。

## 7. 阶段六：设计增强策略

### 7.1 候选增强方法

噪声退化：

- Non-Local Means（NLM）；
- BM3D，可作为扩展方法。

模糊退化：

- Wiener 去卷积；
- 轻度锐化。

低对比度退化：

- CLAHE；
- Gamma 校正。

混合退化：

```text
去噪 -> 对比度增强 -> 轻度锐化
```

### 7.2 三组核心策略

```text
无增强：退化图像直接分割
固定增强：所有退化图像使用统一增强流程
自适应增强：根据退化识别结果选择增强方法
```

硬决策形式：

```text
E(I_d) = E_c(I_d)
```

概率融合形式：

```text
I_e = w1 E_noise(I_d) + w2 E_blur(I_d) + w3 E_contrast(I_d)
```

其中 `w1`、`w2`、`w3` 来自退化分类器的类别概率。

## 8. 阶段七：医学图像分割

为保持模型可解释性，主实验采用传统眼底血管分割方法。

推荐流程：

```text
读取增强图像
-> 提取绿色通道
-> CLAHE 局部增强
-> Frangi vesselness / Top-hat / Sobel
-> 阈值分割
-> 形态学开闭运算
-> 去除小连通区域
-> 输出血管预测 mask S
```

需要分别对以下输入进行分割：

- 未增强退化图像；
- 固定增强图像；
- 自适应增强图像。

最终与人工标注 `G` 比较。

### 10.3 分割性能实验

核心结果表：

| 方法 | Dice | IoU | Precision | Recall | HD |
|---|---:|---:|---:|---:|---:|
| 无增强 |  |  |  |  |  |
| 固定增强 |  |  |  |  |  |
| 自适应增强 |  |  |  |  |  |

### 10.4 不同退化类型实验

分别统计：

- 噪声退化下的分割提升；
- 模糊退化下的分割提升；
- 低对比度退化下的分割提升；
- 混合退化下的分割提升。

### 10.5 消融实验

- 去掉退化识别，所有图像使用固定增强；
- 去掉去噪模块；
- 去掉对比度增强模块；
- 去掉去模糊模块；
- 只使用 PSNR/SSIM，不使用分割指标；
- 使用真实退化标签选择增强，作为策略上界。

### 10.6 鲁棒性实验

- 不同噪声强度；
- 不同模糊核大小；
- 不同对比度压缩程度；
- 不同混合退化组合；
- 不同眼底图像样本。

## 16. 项目总结

本项目首先从清晰眼底图像构造可控退化，然后利用可解释统计特征识别退化类型，根据识别结果选择对应增强策略，最后通过图像质量指标和医学血管分割指标验证自适应增强的有效性。

```text
退化建模是基础，退化识别是决策依据，
自适应增强是核心，医学分割指标是最终验证。
```
